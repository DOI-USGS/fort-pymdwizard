Windows Installation¶

Download a copy of the pre-compiled Windows installer from ScienceBase:

metadatawizard_setup_2.0.0.exe.


Once the download is complete, click on the file in your web browser to begin installation.

Installation does not require elevated (admin) privileges. Most users will want to select the defaults as they step through the installation procedure in the window that pops up. When the installation finishes, the application can be launched from the duck icon () on the user’s desktop or start menu.
